# NEXORA Logo Branding Project

## Overview

NEXORA is a fictional cloud technology startup brand created as part of a Logo Design Project. The project focuses on building a modern and minimalist brand identity using Canva.

## Project Goals

* Design a professional startup logo
* Create multiple logo variations
* Develop a consistent visual identity
* Present the logo using real-world mockups

## Tools Used

* Canva

## Deliverables

* Main Logo Design
* Black & White Logo Version
* Icon-Only Logo
* Mockup Presentation

## Design Style

* Minimal
* Modern
* Corporate
* SaaS/Cloud Technology Inspired

## Color Palette

* Dark Navy Blue
* Dark Black
* White
* Gray
* Accent Blue

## Typography

* Bold modern sans-serif typography
* Clean spacing and hierarchy

## Project Outcome

This project helped in understanding:

* Brand identity creation
* Typography selection
* Logo presentation techniques
* Minimalist design principles
